# Build introduction

we need to install something:

For Ubuntu and Debian:
```shell
sudo apt update && sudo apt install -fy python3 python3-pip python3-venv 
```
For Fedora:
```shell
sudo dnf upgrade --refresh && sudo dnf install -y python3 python3-pip python3-venv 
```
